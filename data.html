    The table below shows the underlying data we used in our project. We would like to thank Project Tycho for the use of the data.

    Data Information:

    We utilized a pre compiled data set containing only information on domestic West Nile cases. We abbreviated the raw data to only include data from 2010-2017. Records before 2010 contained too many instances of missing data and missing reporting periods. This dataset also does not clarify whether the case was domestically obtained or imported from a traveler coming back into the country
