    • Project Information:

    Welcome to a visualization website for the case reports of West Nile Virus in the United States, from 2007 - 2017. As you will see, there is an interactive map of the United States, which shows the number of cases in each state, incremented by year. There is also a bar chart that shows the number of total cases per state in the ten year reporting period, along with a pie chart which shows how much each state contributes to the overall case number for the country.

    If you are unfamiliar with West Nile Virus, please read below for some basic information regarding the disease and its impact on society.
